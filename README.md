# Serenity Engine

Serenity is a graphics engine made using [rust](https://www.rust-lang.org/) and [wgpu](https://wgpu.rs/).

## Quickstart

```
cargo run -r -p editor
```

## Prerequisites (web)

* [trunk](https://trunkrs.dev/)
